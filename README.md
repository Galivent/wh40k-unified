# WH40K Unified — Foundry VTT Game System

A unified game system for all WH40K TTRPG lines: **Dark Heresy 2e**, **Rogue Trader**, and **Only War**.

## Installation
Paste this URL into Foundry Setup → Game Systems → Install System → Manifest URL:
```
https://raw.githubusercontent.com/Galivent/wh40k-unified/main/system.json
```

## Actor Types
- **Character** — Full DH2e/RT/OW character with characteristics, skills, wounds, fate, corruption, insanity
- **NPC** — Simplified character sheet for enemies and allies
- **Void Ship** — Rogue Trader void combat vessel
- **Vehicle** — Only War ground vehicles
- **Imperial Knight** — Knight walkers
- **Fighter / Bomber** — Dogfighting aircraft

## Item Types
- Weapon, Armour, Gear, Talent, Trait, Psychic Power, Ship Component, Vehicle Component

## Core Mechanic
d100 roll-under with degrees of success/failure. Righteous Fury on 1-5. Fumble on 96-100.
